These badges are available for **members of The Matrix.org Foundation** to showcase their support for the ecosystem. Whether you are an Associate, Ecosystem, Individual, Silver, Gold, or Platinum member, you can use these assets on your website, GitHub profiles, or marketing materials.

[Not a member yet? Learn more about our membership tiers and benefits here.](@/foundation/members/index.md)

### How to use them

- **Link it back:** If you display a badge on your site, we encourage you to link the image to our [Members Page](@/foundation/members/index.md). This helps verify your support and helps others discover the Foundation.
- **Visual Integrity:** Please do not stretch, rotate, or modify the colors of the badges. Do not alter the aspect ratio of the badge.

**Example HTML snippet:**

```html
<a href="https://matrix.org/foundation/members/">
    <img
        src="path/to/matrix-silver-badge.svg"
        alt="Matrix.org Silver Member Badge"
    />
</a>
```

## Commonly Used Terms

The following table lists some terms that either are wordmarks of The Matrix.org Foundation or simply commonly used in the context of Matrix, alongside their canonical spelling.

| Official Term | Variants | Explanation |
|---|---|---|
| the Matrix protocol | Matrix | The open protocol for decentralised, secure communications. The proper noun Matrix is capitalised. Do not spell as `[matrix]` in written text; the brackets are only added in the logos and icons. |
| the public Matrix network | Matrix | The public communication network using the Matrix protocol ("on Matrix", not "in (the) Matrix"). The proper noun Matrix is capitalised. Do not spell as `[matrix]` in written text; the brackets are only added in the logos and icons. |
| the Matrix ecosystem | | See <https://matrix.org/foundation/about/#mission>. |
| The Matrix.org Foundation C.I.C. | The Matrix.org Foundation, (the) Foundation, Matrix.org, (The Matrix Foundation) | The organisation stewarding the above technology. Each word including The is capitalised due to abbreviating the proper name. |
| matrix.org homeserver | matrix.org (if context permits) | The Matrix [homeserver instance of the Foundation](https://matrix.org/homeserver/about/) which uses the [server name](https://spec.matrix.org/v1.17/appendices/#server-name) matrix.org. Lower case (like the name giving domain), unless [title cased](@/homeserver/about.md). |
| matrix.org website | matrix.org (if context permits) | The website of the Foundation at <https://matrix.org>. Lower case (like the name giving domain), unless title cased. |
| The Matrix Conference | | The [annual conference](https://conference.matrix.org/) hosted by the Foundation. Capitalised. The canonical URL is <https://conference.matrix.org/> with YEAR.matrix.org serving as archive/semi-permalink. |
| Governing Board | GB | The Foundation's [Governing Board](https://matrix.org/foundation/governing-board/) is an organ enabling ecosystem participation. Capitalised. |
| Committee | | [Committees](https://matrix.org/foundation/governing-board/committees/) are subsets of the Governing Board. Capitalised, including specific ones ("Community Committee"). |
| Working Group | WG | [Working Groups](https://matrix.org/foundation/working-groups/) are an instrument of the Governing Board. Capitalised, including specific ones ("Events Working Group"). |
| other organs | | Other organs of the Foundation are similarly capitalised, e.g. "Guardians", "Spec Core Team", "Trust & Safety". |
| `#room:server.tld` | | Pronounced "[hash room on server dot tld](https://matrix.org/blog/wp-content/uploads/2014/09/Introducing-matrix.pdf)". |
| `@user:server.tld` | | Pronounced "[at user on server dot tld](https://matrix.org/blog/wp-content/uploads/2014/09/Introducing-matrix.pdf)". |

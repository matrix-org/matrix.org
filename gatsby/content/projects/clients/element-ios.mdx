The native iOS version of the [Element](/docs/projects/client/element) web client. Element iOS is a glossy client with focus on performance and usability.

This client was previously known as Riot iOS.

The code is available from [GitHub](https://github.com/vector-im/element-ios) and the app is available from the [Apple App Store](https://itunes.apple.com/gb/app/vector.im/id1083446067?mt=8).

You can help test the beta version of Element on [TestFlight](https://testflight.apple.com/join/lCeTuDKM).

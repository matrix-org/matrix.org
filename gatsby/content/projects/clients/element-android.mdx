The native Android version of the [Element](/docs/projects/client/element) web client. Element is a glossy client with focus on performance and usability.

While under development, Element Android was known as RiotX, and once released it replaced Riot Android as the flagship Matrix client for Android.

The code is available from [GitHub](https://github.com/vector-im/element-android), and the app is available from the [Google Play store](https://play.google.com/store/apps/details?id=im.vector.app) and [F-Droid](https://f-droid.org/packages/im.vector.app/).

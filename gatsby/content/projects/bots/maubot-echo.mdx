A [maubot](https://github.com/maubot/maubot) that echoes pings and other stuff.

Most of the ping bots in [#ping:maunium.net](https://matrix.to/#/#ping:maunium.net) are using this plugin.

## Usage
* `!ping` - Reply with "Pong!" and the time it took for the message to reach the bot.
* `!echo <message>` - Reply with the given message

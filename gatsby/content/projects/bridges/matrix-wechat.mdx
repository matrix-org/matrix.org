A Matrix-Wechat puppeting bridge based on [mautrix-go](https://github.com/mautrix/go).

✓ Private chat  
✓ Group chat  
✓ Plain text message  
✓ Most message media (photo, video, voice, file, location)  
✓ Reply and mention  
✓ Double puppeting  

Current feature list at <https://github.com/duo/matrix-wechat#features--roadmap>

A Matrix-Instagram DM puppeting bridge. Written in Python using [mauigpapi](https://github.com/tulir/mautrix-instagram/tree/master/mauigpapi) and [mautrix-python](https://matrix.org/docs/projects/sdk/mautrix-python.html).

✓ 1:1 chats  
✓ Group chats  
✗ Stories  
✗ Feed  
✓ Plain text  
✗ Formatted messages  
✓ Message reactions  
✓ Message redactions  
✓ Photos, videos and gifs  
✗ Arbitrary files  
✓ Typing notifications and read receipts  
✗ Presence  
✓ Double puppeting  
✓ Backfilling history and missed messages  
✓ End-to-bridge encryption

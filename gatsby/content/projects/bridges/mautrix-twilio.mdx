A Matrix-Twilio relaybot bridge for WhatsApp Business. Can also be used for Twilio SMS bridging with small modifications. Written in Python using [mautrix-python](https://matrix.org/docs/projects/sdk/mautrix-python.html).

✓ 1:1 chats  
✗ Group chats  
✓ Plain text and formatted messages  
✓ Images, videos and files

A Matrix-Signal puppeting bridge. Written in Python using [mautrix-python](https://matrix.org/docs/projects/sdk/mautrix-python.html).
Communicates with Signal using [signald](https://gitlab.com/signald/signald).

✓ 1:1 chats  
✓ Group chats  
✓ Plain text  
✗ Formatted messages  
✓ Photos, gifs, voice notes and other files  
✓ Stickers (only Signal->Matrix)  
✓ Read receipts  
✓ Typing notifications (only Signal->Matrix)  
✓ Double puppeting  
✓ End-to-bridge encryption

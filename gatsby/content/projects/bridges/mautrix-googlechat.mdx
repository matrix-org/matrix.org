A Matrix-Google Chat puppeting bridge. Written in Python using [a fork of hangups](https://github.com/mautrix/googlechat/tree/master/maugclib) and [mautrix-python](https://matrix.org/docs/projects/sdk/mautrix-python.html).

✓ 1:1 chats  
✓ Group chats  
✓ Plain text  
✓ Threads  
✓ Formatted messages  
✓ Images  
✗ Other file types  
✓ Typing notifications and read receipts  
✓ Double puppeting  
✗ Backfilling history and missed messages  
✓ End-to-bridge encryption

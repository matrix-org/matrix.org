A Matrix-Slack puppeting bridge. Written in Go using a fork of [slack-go](https://github.com/slack-go/slack)

✓ 1:1 and group DMs  
✓ Channels  
✓ Slack connect channels  
✓ Plain text and formatted messages  
✓ Threads  
✓ Message redaction/deletion  
✓ Reactions  
✗ Custom emojis (Slack -> Matrix only)  
✓ Own read status sync  
✓ Double puppeting  
✗ Backfilling history  
✗ Relaybot mode  
✓ End-to-bridge encryption

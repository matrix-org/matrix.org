A WhatsApp Matrix-Gupshup bridge. Written in Python using [mautrix-python](https://matrix.org/docs/projects/sdk/mautrix-python.html).

✓ 1:1 chats  
✗ Group chats  
✗ User Profiles  
✓ Plain text messages  
✓ Formatted messages  
✓ Most message media (photos, videos, GIFs, files)  
✓ Read receipts  
✗ Presence  
✗ Double puppeting  
✗ Backfilling history and missed messages  
✗ End-to-bridge encryption
